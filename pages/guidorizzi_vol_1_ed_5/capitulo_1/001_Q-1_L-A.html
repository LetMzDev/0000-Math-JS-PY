Exercício 1.2

1) Resolva as inequações
A) \( 3x + 3 < x + 6 \)

Passo 1: Subtraia \( 3 \) de ambos os lados

Subtraímos \( 3 \) para isolar o termo com \( x \):

\[ 3x + 3 - 3 < x + 6 - 3 \]

Simplificando:

\[ 3x < x + 3 \]

Passo 2: Subtraia \( x \) de ambos os lados

Subtraímos \( x \) para isolar os termos com \( x \) no lado esquerdo:

\[ 3x - x < x - x + 3 \]

Simplificando:

\[ 2x < 3 \]

Passo 3: Divida ambos os lados por 2

Dividimos por 2 para resolver para \( x \):

\[ \frac{2x}{2} < \frac{3}{2} \]

Simplificando:

\[ x < \frac{3}{2} \]

Solução Final:

\[ x < \frac{3}{2} \]

Notação de intervalo: \( \left(-\infty, \frac{3}{2}\right) \)
